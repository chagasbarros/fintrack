# FinTrack

O FinTrack é um sistema de controle de finanças pessoais via console. 

O projeto foi feito com Java puro, com foco na prática de POO e
estruturas básicas, e poderá ser expandido futuramente com relatórios
gráficos, persistência em banco de dados e integração com APIs.

## Tecnologias

- Java
- Programação Orientada a Objetos
- Collections
- Exceptions personalizadas

## Funcionalidades

- Cadastro de receitas e despesas
- Listagem de transações
- Cálculo de saldo
- Remoção de transações
